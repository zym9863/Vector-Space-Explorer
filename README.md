# 向量空间探索器 (Vector Space Explorer)

一个基于Three.js的交互式向量可视化和运算工具，帮助用户直观理解向量的几何意义和代数运算。

## 功能特性

### 🎯 核心功能

#### 1. 交互式向量可视化与运算
- **向量创建与修改**: 在2D/3D空间中实时创建和调整向量
- **向量加法可视化**: 通过平行四边形法则展示向量加法的几何意义
- **向量减法可视化**: 展示向量减法的几何表示
- **标量乘法**: 演示标量对向量大小和方向的影响

#### 2. 向量几何意义解析
- **点积可视化**: 
  - 实时计算并显示点积数值
  - 展示点积与向量投影的关系
  - 可视化垂直投影线
- **叉积可视化**: 
  - 显示叉积向量的方向（右手定则）
  - 展示叉积的大小（平行四边形面积）
  - 3D空间中的完整叉积表示
- **投影可视化**: 
  - 一个向量在另一个向量上的投影
  - 清晰的几何表示和数值计算
- **夹角显示**: 
  - 实时计算向量间夹角
  - 弧形可视化表示

### 🛠 技术特性

- **2D/3D模式切换**: 支持在二维和三维空间中进行向量操作
- **实时计算**: 所有运算结果实时更新
- **交互式控制**: 
  - 鼠标控制相机视角
  - 数值输入框精确调整向量
  - 颜色自定义
- **视觉增强**: 
  - 高质量渲染
  - 平滑动画效果
  - 清晰的标签和数值显示

## 使用说明

### 快速开始

1. 直接在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖，即开即用

### 界面操作

#### 向量控制
- **坐标输入**: 在左侧面板中直接输入X、Y、Z坐标值
- **颜色设置**: 点击颜色选择器自定义向量颜色
- **模式切换**: 点击2D/3D按钮切换显示模式

#### 向量运算
- **加法**: 点击"显示加法(A + B)"按钮查看向量加法结果
- **减法**: 点击"显示减法(A - B)"按钮查看向量减法结果  
- **标量乘法**: 
  1. 点击"标量乘法"按钮
  2. 设置标量值
  3. 选择要操作的向量

#### 几何运算
- **点积**: 点击"点积可视化"查看投影关系
- **叉积**: 点击"叉积可视化"查看垂直向量
- **投影**: 点击"投影可视化"查看向量投影
- **夹角**: 点击"夹角显示"查看角度弧线

#### 显示选项
- **网格**: 切换坐标网格显示
- **坐标轴**: 切换XYZ轴显示
- **原点**: 切换原点标记显示
- **标签**: 切换向量标签显示

### 3D控制

- **旋转**: 按住鼠标左键拖动
- **缩放**: 鼠标滚轮或双指缩放
- **平移**: 按住鼠标右键拖动

## 技术实现

### 技术栈
- **Three.js**: 3D图形渲染引擎
- **HTML5/CSS3**: 现代Web标准
- **ES6+ JavaScript**: 模块化编程

### 关键特性
- **ArrowHelper**: 使用Three.js的箭头辅助对象表示向量
- **动态几何体**: 实时生成平行四边形、投影线等几何图形
- **Canvas纹理**: 动态生成向量标签和数值显示
- **材质系统**: 使用发光材质增强视觉效果

### 核心算法
- **向量运算**: 基于Three.js Vector3类的标准向量数学
- **点积计算**: `A·B = |A||B|cosθ`
- **叉积计算**: `A×B = |A||B|sinθ n̂`
- **投影计算**: `proj_B(A) = (A·B/|B|²)B`
